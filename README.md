# Stock Price Analysis Using Python

This repository contains a Python-based analysis of stock price data using a Jupyter Notebook.  
The project focuses on exploring historical stock prices, identifying trends, and generating insights through data analysis and visualisation techniques.

---

##  Project Overview

Stock markets generate large volumes of time-series data. This project applies data analysis techniques to historical stock price data in order to:
- Understand price movements over time
- Identify trends and patterns
- Perform exploratory analysis and visualisation
- Support data-driven insights

The notebook documents the full analytical workflow, ensuring clarity and reproducibility.

---

## Repository Contents

- `stock_price_analysis.ipynb`  
  Main Jupyter Notebook containing:
  - Data loading and cleaning
  - Exploratory Data Analysis (EDA)
  - Time-series visualisation
  - Statistical analysis and indicators
  - Interpretation of results

All outputs are preserved in the notebook.
